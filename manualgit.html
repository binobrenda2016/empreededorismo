  • Git commit –amend: Para alterar o commit>

  • Git clone Esse comando Git cria uma cópia exata de um repositório já existente. Então… quando usar git init e quando usar git clone? O git clone é mais avançado, uma vez que ele mesmo executa um comando git init internamente. Além disso, ele verifica todo o conteúdo do projeto.
  • Git add Esse comando Git adiciona os arquivos especificados de código ao seu repositório, sejam arquivos novos ou arquivos anteriores que foram alterados. Oferece diferentes possibilidades de sintaxe.
  • Git branch É comum na maior parte do tempo possuir múltiplas variações em seu repositório Git, chamadas de branches (“ramificações”). A grosso modo, um branch é um caminho independente de desenvolvimento, uma alternativa. A princípio pode parecer fácil se perder em diversos caminhos, mas o comando git branch facilita o gerenciamento de tudo.

    Empreendorismo Social

    Empreendedorismo social é um conceito que se refere ao processo de identificação de um problema social e à criação de soluções inovadoras e sustentáveis para resolvê-lo⁴. Ele é uma forma de empreendedorismo que busca desenvolver produtos e serviços que impactem a sociedade, ajudando a solucionar os problemas enfrentados por ela

    Esses empreendimentos são criados para produzir bens e serviços em benefício da comunidade, promovendo a melhoria na qualidade de vida das pessoas³. Eles podem fazer isso oferecendo capacitação, emprego, oportunidades de tratamento de saúde, atuando na preservação ao meio ambiente, entre outras frentes

    .

    contrário dos empreendedores tradicionais, que buscam maximizar o lucro, os empreendedores sociais buscam maximizar o impacto positivo em uma comunidade ou em uma sociedade⁴. Enquanto o empreendedor “comum” cria um negócio para obter lucro, atendendo uma demanda do mercado, o empreendedor social toma uma iniciativa para resolver um problema, atender uma necessidade social

    As competências fundamentais para que o empreendedor social tenha um bom desempenho são: visionário, senso de responsabilidade, senso de solidariedade, sensibilidade diante de questões sociais, persistência, liderança, consciência e saber usar forças latentes e regenerar forças pouco usadas

Geração de Idéias

A geração de ideias é um processo criativo que as empresas usam para gerar novas ideias, sejam elas tangíveis ou intangíveis. Envolve a coleta de ideias, pesquisa, teste, edição ou revisão e, por fim, a implementação do plano.

Existem muitas técnicas que podem ser utilizadas para a geração de ideias. Algumas delas incluem:

1. **Mindmapping**: Uma técnica visual que ajuda a organizar informações e ideias de maneira estruturada

¹.

2. **Pensamento de atributo**: Esta técnica envolve a análise de um problema em seus componentes individuais para gerar novas ideias

s¹.

3. **Brainstorming reverso**: Em vez de perguntar como resolver um problema, pergunte como poderia causar o problema

¹.

4. **Encontrar inspiração**: Isso pode envolver a busca de novas ideias em lugares inesperados ou a aprendizagem com o que outras empresas estão fazendo

¹.

a5. **Uso de uma ferramenta online**: Existem várias ferramentas online disponíveis que podem ajudar no processo de geração de ideias

¹.

Essas técnicas podem ser usadas individualmente ou em combinação, dependendo do problema específico que você está tentando resolver. Lembre-se, o objetivo é gerar uma ampla gama de ideias e depois refinar e desenvolver as mais promissoras.
