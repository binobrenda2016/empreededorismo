Tipos de Gits

  • Feat: Indica que seu trecho de código está incluindo um novo recurso
  • Fix: Indica que seu trecho de código commitado está solucionando um problema (bug fix)
  • Docs: Indica que houveram mudanças na documentação
  • Style: Indica que houveram alterações referentes a formatações de código
  • Refactor: Referem-se a mudanças devido a refatorações que não alterem sua funcionalidade
  • Build: São utilizados quando são realizadas modificações em arquivos de build e dependências
  • Test: São utilizados quando são realizadas alterações em testes

Comandos intemediário Git

  • Git remote add : Este comando estabelece uma conexão entre seu repositório local e um repositório remoto
  • Git push -u remote branch: Este comando serve para subir suas modificações para um repositório remoto conectado anteriormente com git remote
  • Git fetch: Quando você precisa baixar as mudanças criadas por outros membros do seu projeto colaborativo, você precisa do comando Git fetch
  • Git commit –amend: Para alterar o commit>
  • Git diff –staged: Compara a área de teste com o último commit
  • Git clone Esse comando Git cria uma cópia exata de um repositório já existente. Então… quando usar git init e quando usar git clone? O git clone é mais avançado, uma vez que ele mesmo executa um comando git init internamente. Além disso, ele verifica todo o conteúdo do projeto.
  • Git add Esse comando Git adiciona os arquivos especificados de código ao seu repositório, sejam arquivos novos ou arquivos anteriores que foram alterados. Oferece diferentes possibilidades de sintaxe.
  • Git branch É comum na maior parte do tempo possuir múltiplas variações em seu repositório Git, chamadas de branches (“ramificações”). A grosso modo, um branch é um caminho independente de desenvolvimento, uma alternativa. A princípio pode parecer fácil se perder em diversos caminhos, mas o comando git branch facilita o gerenciamento de tudo.

    • Empreededorismo

    Empreendedorismo é um termo que se refere à capacidade de criar, desenvolver e gerenciar negócios ou projetos inovadores, que buscam solucionar problemas ou aproveitar oportunidades da sociedade. O empreendedorismo pode ter diferentes tipos, como o corporativo, o social, o digital, o sustentável, entre outros. O empreendedorismo é importante para a economia, pois gera riqueza, emprego e inovação. Além disso, o empreendedorismo pode contribuir para a melhoria da qualidade de vida das pessoas, por meio de soluções que atendam às suas necessidades e desejos Empreededorismo

      Tipos de Empreendedorismo

      Existem vários tipos de empreendedorismo, que se diferenciam pelo perfil, pelo objetivo e pela forma de atuação dos empreendedores. Alguns dos principais tipos são:

    • Empreendedorismo público**: é o que busca soluções inovadoras para os problemas da sociedade, por meio de políticas, programas ou projetos públicos
    • Empreendedorismo corporativo**: é o que ocorre dentro de uma organização já estabelecida, quando os colaboradores desenvolvem novos produtos, serviços ou processos que geram valor para a empresa e para o mercado.
    • **Empreendedorismo cooperado**: é o que se baseia na união de pessoas que compartilham interesses e recursos para criar e gerir um empreendimento coletivo, com benefícios mútuos.
    • **Empreendedorismo inesperado**: é o que surge de uma situação adversa, como uma demissão, uma crise ou uma necessidade urgente, que leva a pessoa a buscar uma alternativa de renda por meio de um negócio próprio.
    • **Empreendedorismo informal**: é o que acontece sem o registro legal ou a formalização do negócio, geralmente por falta de conhecimento, recursos ou incentivos. Esse tipo de empreendedorismo pode trazer riscos e limitações para o crescimento do negócio.
    • **Empreendedorismo do conhecimento**: é o que utiliza o conhecimento científico, tecnológico ou criativo como matéria-prima para criar novas soluções, produtos ou serviços. Esse tipo de empreendedorismo requer alto grau de inovação e investimento em pesquisa e desenvolvimento.
    • **Empreendedorismo normal**: é o que segue os padrões tradicionais de gestão e operação de um negócio, sem grandes diferenciais competitivos ou inovações disruptivas. Esse tipo de empreendedorismo pode ter sucesso se houver demanda e qualidade no mercado
    • **Empreendedorismo serial**: é o que caracteriza os empreendedores que criam vários negócios ao longo da vida, buscando novas oportunidades e desafios. Esse tipo de empreendedorismo requer visão estratégica, adaptabilidade e disposição para assumir riscos.
    • **Empreendedorismo herdeiro**: é o que acontece quando uma pessoa assume a gestão de um negócio familiar ou herdado, podendo manter, expandir ou transformar o legado recebido. Esse tipo de empreendedorismo requer comprometimento, responsabilidade e respeito pela história do negócio.
    • **Empreendedorismo interno**: é o que ocorre quando uma pessoa cria um negócio dentro de outro negócio, aproveitando as estruturas, os recursos e as parcerias já existentes. Esse tipo de empreendedorismo requer autonomia, alinhamento e colaboração entre as partes envolvidas.
    • - **Empreendedorismo individual**: é o que se refere ao profissional autônomo que presta serviços por conta própria, sem vínculo empregatício com nenhuma empresa. Esse tipo de empreendedorismo requer organização, disciplina e capacitação constante.
    • - **Empreendedorismo franqueado**: é o que se baseia na aquisição de uma franquia, que é um modelo de negócio já testado e padronizado por uma empresa matriz. Esse tipo de empreendedorismo requer investimento inicial, cumprimento das normas e pagamento de taxas à franqueadora.
    • - **Empreendedorismo social**: é o que visa gerar impacto social positivo por meio de iniciativas que atendam às necessidades ou aos problemas de uma comunidade ou de um grupo vulnerável. Esse tipo de empreendedorismo requer sensibilidade, engajamento e sustentabilidade financeira.
    • **Empreendedorismo digital**: é o que utiliza a internet como plataforma para criar e divulgar produtos ou serviços online, aproveitando as oportunidades e as facilidades do mundo digital. Esse tipo de empreendedorismo requer criatividade, atualização e conhecimento técnico.
